Tips

  • Scala Worksheet の結果表示量の変更について
    1. [Eclipe] -> [環境設定] -> [Scala Worksheet]
    2. Output character limit per statement の値を変更する
  • キーボードショートカットの設定について
    1. [Eclipe] -> [環境設定] -> [一般] -> [キー]
    2. 好みのショートカットに編集する
  • エディタのフォントの大きさを変更する
    1. エディタのうえで右クリック
    2. [設定」-> [一般] -> [外観] -> [色とフォント]
    3. 好みのフォントに変更する

エラー

  • ファイルをセーブしても Worksheet がプログラムを評価しなくなった
    • Thunderbird など他のアプリケーションを起動している場合はメモリが少 な動作が遅くなる場合がある．Thunderbird その他アプリを終了してみよ う．
    • Worksheet を一度閉じて，もう一度読み直してみよう．
    • 他の場所でエラー (赤バッテン) が出てる場合は，そこで処理が引っかか る場合がある．他の場所のエラーを解決してみよう．
  • よく分からないエラー (赤バッテン) が出る時の対応
    • エラーが出ているプロジェクトを右クリックして [更新] してみる
    • Mac OSX の resourc fork file や eclipse のキャッシュが影響している可能性があるので以下の手順で削除してみる
      1. まず Eclipse を閉じる
      2. 以下のコマンドを実行 (コピー&ペーストしましょう♪) 
    cd                                     # ホームディレクトリへ移動
cd ./prolang                           # prolang ディレクトリへ移動
find . -name "._*"                     # 消したいファイルの表示を確認
find . -name "._*" | xargs rm -rf      # それらのファイルを削除
find . -name ".cache"                  # 消したいファイルの表示を確認
find . -name ".cache" | xargs rm -rf   # それらのファイルを削除
    • clean オプションをつけて Eclipse を起動してみる Eclipse はデフォルトでは色々なキャッシュされた情報を使って起動する．clean オプションをつけると多少時間は余計にかかるがそれらを使わずに起動する．
      1. 以下のコマンドを実行 (コピー&ペーストしましょう♪) 
    nohup /Applications/eclipse/Eclipse.app/Contents/MacOS/eclipse -clean &
  • Scala Worksheet への日本語入力でエラーが出る場合
    • [Eclipse] -> [環境設定] -> [ワークスペース] -> [テキスト・ファイル・エンコード] で [その他] にチェックを入れ UTF-8 を指定する
