1 講義資料ファイル week02 のダウンロードと展開

  • 操作
    1. ここ をクリックして講義資料のファイル week02.tar.gz をダウンロードする.
    2. 以下のコマンドを実行 (コピー&ペーストしましょう♪) 
      cd                                     # ホームディレクトリへ移動
mv ~/Downloads/week02.tar ./prolang    # prolang.tar をホームディレクトリへ移動
tar xvzf week02.tar                    # prolang.tar を展開
      • (注意) Safari のデフォルト設定ではダウンロードしたファイルが自動的に week02.tar へと解凍されます．他のブラウザを使用しているなどで解凍されない場合は上記の説明の week02.tar を week02.tar.gz に変更して実行してください．

2 week02 を読み込む

  1. Eclipse を起動
  2. 無事にプロジェクトがインポートできたら，左側にある [プロジェクト・エクスプローラー] の中にある [week02] -> [/src/main/scala/] -> [listPractice] を選択してみる．
  3. week02.sc が出てくるのでダブルクリック
  4. test.sc では書いたコードが1行1行評価される．試しに 1+3 と入力して command + s を押すと計算結果である 4 が右側に表示される

3 Scala JUnit Test でテストする

  1. 左側にある [プロジェクト・エクスプローラー] の中にある [week02] -> [/src/test/scala/] を選択する (青く反転する)．
  2. 画面一番上のメニューの [実行] -> [実行(S)] -> [Scala JUnit Test] をクリック
  3. テスト結果が JUnit というタブに表示される
