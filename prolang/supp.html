第2週

講義資料ファイル week02 のダウンロードと展開

  1. ここ をクリックして講義資料のファイル week02.tar をダウンロードする.
  2. 以下のコマンドを実行 (コピー&ペーストしましょう♪) 
cd                                     # ホームディレクトリへ移動
mv ~/Downloads/week02.tar ./prolang/    # week02.tar をホームディレクトリへ移動
cd ./prolang
tar xvzf week02.tar                    # week02.tar を展開

week02 を読み込む

  1. Eclipse を起動
  2. プロジェクトをインポート
    1. [ファイル] -> [インポート]
    2. インポート画面が開くので，[一般] -> [既存プロジェクトをワークスペースへ] を選択し，[次へ] をクリック．
    3. ルートディレクトリの選択をチェックし，参照ボタンを押して prolang ディレクトリの下にある week02 を選択する．/home*/ユーザ名/prolang/week02 と表示される．
    4. [完了] をクリック
  3. 無事にプロジェクトがインポートできたら，左側にある [プロジェクト・エクスプローラー] の中にある [week02] -> [/src/main/scala/] -> [listPractice] を選択してみる．
  4. week02.sc が出てくるのでダブルクリック
  5. test.sc では書いたコードが1行1行評価される．試しに 1+3 と入力して command + s を押すと計算結果である 4 が右側に表示される

余分なファイル (Eclipse に影響を与えている??) の削除

find . -name "._*"                     # 消したいファイルの表示を確認
find . -name "._*" | xargs rm -rf      # それらのファイルを削除
find . -name ".cache"                  # 消したいファイルの表示を確認
find . -name ".cache" | xargs rm -rf   # それらのファイルを削除

Eclipse の操作について

  • ソースフォルダの作成
    • ソースフォルダを作成したいプロジェクトを選択
    • 右クリックして [新規] -> [その他] -> [Java] -> [ソースフォルダ] を選択して [次へ].
    • 名前を入力して終了
  • パッケージの作成
    • パッケージを作成したいプロジェクトを選択
    • 右クリックして [新規] -> [その他] -> [Java] -> [パッケージ] を選択して [次へ].
    • 名前を入力して終了
  • Scala Worksheet の作成
    • Scala Worksheet を作成したい場所を選択
    • 右クリックして [新規] -> [その他] -> [Scala Wizards] -> [Scala Worksheet] を選択して [次へ].
    • 名前を入力して終了
  • Scala Object の作成
    • Scala Object を作成したい場所を選択
    • 右クリックして [新規] -> [その他] -> [Scala Wizards] -> [Scala Object] を選択して [次へ].
    • 名前を入力して終了
