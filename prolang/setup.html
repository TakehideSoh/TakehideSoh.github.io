1 講義資料ファイル prolang.tar.gz のダウンロードと展開

  • 操作
    1. ここ をクリックして講義資料のファイル prolang.tar.gz をダウンロードする.
    2. 以下のコマンドを実行 (コピー&ペーストしましょう♪) 
      cd                               # ホームディレクトリへ移動
mv ~/Downloads/prolang.tar ./    # prolang.tar をホームディレクトリへ移動
tar xvzf prolang.tar             # prolang.tar を展開
      • (注意) Safari のデフォルト設定ではダウンロードしたファイルが自動的に prolang.tar へと解凍されます．他のブラウザを使用しているなどで解凍されない場合は上記の説明の prolang.tar を prolang.tar.gz に変更して実行してください．
  • 確認
    1. 以下のコマンドの実行結果を確認する 
      cd prolang                       # prolang ディレクトリへ移動
pwd
    2. チェック! 以下のように出力されればOK!
      /home*/ユーザ名/prolang
  • 確認
    1. 以下のコマンドの実行結果を確認する 
      echo $PATH                       # パスの情報を確認
    2. チェック! 特にパスを変えてな ければ以下のように出力される
      • home4/soh/prolang/scala-2.10.2/bin:/usr/bin:/bin:/usr/sbin:/sbin:/usr/local/bin:/opt/local/bin:/opt/local/sbin/:/usr/local/mysql/bin/:/opt/X11/bin:/usr/X11/bin:/usr/local/ncbi/blast/bin

2 Scala へのパスを通す

  • 操作
    1. 以下のコマンドを実行 
      cd
cd prolang/scala-2.10.2/bin
pwd                         # "出力確認" /home*/ユーザ名/prolang/scala-2.10.2/bin
E=`pwd`; echo export PATH=$E:\$PATH >> ~/.bash_profile
source ~/.bash_profile
  • 確認
    1. 以下のコマンドの実行結果を確認する 
      scala -version
    2. チェック! 以下のように表示されればOK!
      Scala code runner version 2.10.2 – Copyright 2002-2013, LAMP/EPFL

3 Eclipse の起動と Scala IDE のインストール

  1. Eclipse を起動
    1. 画面右下にある [アプリケーション] をクリックするとアプリケーショ ンフォルダの内容が表示されるので，eclipse と書かれた青紫色のアイコンを Dock へドラッグする
    2. Dock へ Eclipse が配置されたらダブルクリックして Eclipse を 起動する
  2. ワークスペース・ランチャーが立ち上がるので /home*/ユーザ名/prolang/ を指定する
  3. [ヘルプ] -> [新規ソフトウェアのインストール] を選択
  4. 開いたウィンドウの [作業対象] のところに以下をコピーして，ペースト
    http://download.scala-ide.org/sdk/e38/scala210/stable/site
  5. Scala IDE の項目が白い枠内に表示されるので，[すべて選択] をチェックし，[次へ] をクリック.
    (表示されない場合は [作業対象] のところで Return キーを押してみる)
  6. インストール詳細画面になる．[次へ].
  7. ライセンスをレビュー画面になる．使用条件の条項に同意しますをチェックして，[完了].
  8. Scala IDE のインストールが始まる．セキュリティー警告が出るが [ok] をクリックして進める．
  9. インストールが完了したら再起動するかどうか聞かれるので，[はい] をクリック．
  10. 再起動すると Run Scala Setup Dianostics が出る．[はい] をクリック．
  11. Setup Dianostics 画面の Scala JDT の項目で [推奨されたデフォルト設定を使用する] をチェックして [OK] をクリック.

4 Eclipse にプロジェクトをインポートする

  1. [ファイル] -> [インポート]
  2. インポート画面が開くので，[一般] -> [既存プロジェクトをワークスペースへ] を選択し，[次へ] をクリック．
  3. ルートディレクトリの選択をチェックし，参照ボタンを押して prolang ディレクトリの下にある week01 を選択する．/home*/ユーザ名/prolang/week01 と表示される．
  4. [完了] をクリック

5 Scala Worksheet で遊んでみる

  1. プロジェクトがインポートできたら，左側にある [プロジェクト・エクスプローラー] の中にある [week01] -> [/src/main/scala/] -> [listPractice] を選択してみる．
  2. test.sc が出てくるのでダブルクリック
  3. test.sc では書いたコードが1行1行評価される．試しに 1+3 と入力して command + s を押すと計算結果である 4 が右側に表示される
  • Scala Worksheet への日本語入力でエラーが出る場合
    • [Eclipse] -> [環境設定] -> [ワークスペース] -> [テキスト・ファイル・エンコード] で [その他] にチェックを入れ UTF-8 を指定する
    • 参考
      • 本家サイト https://sites.google.com/site/scalajp/home/scala-pitfalls/file\_encoding
      • Java Drive http://www.javadrive.jp/eclipse3/setup/index1.html
