1 基本方針

  • sbt のインストール方法については, 本家のサイト に詳しい解説が記載 されているので，これを参照するのが一番確実．
    • ただし，Windows に関しては本家サイトの方法だと環境変数の設定が 必要になるので, パッケージ管理ソフト chocolatey を使う以下の方法 がおすすめ．
  • 以下では, 上記のサイトを参考にして, Windows10, Ubuntu 18.04, Mac OS 10.11.6 で成功した方法を記載する.

2 Windows における sbt 他のインストール方法

  • Windows で Scala プログラミングするために以下が必要になる．
    • Java SDK (8以降)
    • sbt
  • Windows でこれらのソフトウェアを直接インストールするとパスの 設定などが，面倒なことがある．
  • Windows のパッケージ管理システムである chocolatey を使うと, パスを 自動的に追加してくれるので便利である.
  • 以下では, chocolatey を使ったこれらのインストール方法を説明する．
    • Windows10 で Powershell を使う方法を想定している．

2.1 chocolatey のインストール

https://chocolatey.org/install へアクセスし指示に従う． 2019年3月時点で，以下の方法でインストールできることを確認している (Windows 10)．

  • Powershell (管理者として実行) を開き, 以下のコマンドを実行する． 

    Set-ExecutionPolicy Bypass -Scope Process -Force; iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))
  • Powershell を閉じる

  • (確認) Powershell を開き, 以下のコマンドを実行し, バージョンが表 示されれば成功. 

    > choco

2.2 Java 8 SDK と sbt のインストール (chocolatey を使用)

  • Powershell (管理者として実行) を開き, 以下のコマンドを実行する． 

    > choco install jdk8
  • Powershell を閉じる

  • Powershell (管理者として実行) を開き, 以下のコマンドを実行する． 

    > choco install sbt
  • Powershell を閉じる

2.3 IntelliJ IDEA のインストール (chocolatey を使用)

sbt があれば, scala プログラミングは可能．追加で IntelliJ IDEA を インストールする場合は以下の方法で行う．

  • Powershell (管理者として実行) を開き, 以下のコマンドを実行する． 

    > choco install intellijidea-community
  • Powershell を閉じる

2.4 scala のインストール (chocolatey を使用)

sbt があれば, scala プログラミングは可能．追加で scala を インストールする場合は以下の方法で行う．

  • Powershell (管理者として実行) を開き, 以下のコマンドを実行する． 

    > choco install scala
  • Powershell を閉じる
  • ただし, 2019年3月時点で, chocolatey でインストールできる scala は, 最新版ではなくバージョン 2.11.8 になる.

3.1 sbt のインストール

  • Macports を使ってインストールするのが便利. 

    $ sudo port install sbt

4 Ubuntu Linux (他 Debian系) における sbt 他のインストール方法

4.1 sbt のインストール

  • apt や apt-get を使ってインストールするのが便利.

  • apt-get を使った方法を以下に示す (参考). 

    $ echo "deb https://dl.bintray.com/sbt/debian /" | sudo tee -a /etc/apt/sources.list.d/sbt.list
$ sudo apt-key adv --keyserver hkp://keyserver.ubuntu.com:80 --recv 2EE0EA64E40A89B84B2DF73499E82A75642AC823
$ sudo apt-get update
$ sudo apt-get install sbt
  • Java がないとエラーが出れば以下を行った後に, もう一度上記を実行す る.

4.2 JDK 8 のインストール

  • apt や apt-get を使ってインストールするのが便利. 以下は openjdk をインストールしている. 

    $ sudo apt-get install openjdk-8-jdk
