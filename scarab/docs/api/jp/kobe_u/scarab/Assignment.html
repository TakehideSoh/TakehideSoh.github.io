Assignment

Related Doc: package scarab

case class Assignment(intMap: Map[Var, Int], boolMap: Map[Bool, Boolean]) extends Product with Serializable

Assignment is a case class representing the assignment for integer variables and Boolean variables. The assignment is returned from the jp.kobe_u.scarab.solver.Solver as a solution.

scala> val x = Var("x")
scala> val y = Var("y")
scala> val assignment = Assignment(Map(x->1, y->2), Map.empty)
scala> (x + y).value(assignment)
res0: Int = 3
